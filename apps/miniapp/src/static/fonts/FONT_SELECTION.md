# 字体选型记录

## 选定字体

**方正黑体**

| 字段     | 内容                                                                                         |
| -------- | -------------------------------------------------------------------------------------------- |
| 字体名称 | 方正黑体                                                                                     |
| 英文名   | `FZHei-B01`                                                                                  |
| 来源     | 方正字库官方免费字体说明：https://www.foundertype.com/index.php/About/powerbus.html          |
| 授权口径 | 官方列入可免费用于商业发布的基础字体；小程序内嵌、APP 嵌入或 Web Font 在线调用需另行确认授权 |
| 文件位置 | 当前不提交字体文件                                                                           |

## 使用范围

| 场景                          | 使用字体栈                |
| ----------------------------- | ------------------------- |
| 品牌标题、金额数字、倒计时    | 方正黑体展示字体栈        |
| 等级、badge、入口标题、短标签 | 方正黑体展示字体栈        |
| 长正文、帖子内容、表单说明    | 方正黑体正文栈 + 系统兜底 |
| 错误提示、账本明细            | 方正黑体正文栈 + 系统兜底 |

## Fallback 策略

```css
font-family:
  "FZHei-B01", "方正黑体_GBK", "方正黑体", "FounderType Hei", "PingFang SC", "Hiragino Sans GB",
  "Microsoft YaHei", sans-serif;
```

- 设备未安装方正黑体时，降级到常见中文系统字体，保证小程序文字可读。
- 当前不通过 `@font-face` 打包字体文件，避免误入小程序内嵌授权范围。
- 如后续购买或取得明确的小程序嵌入授权，再将字体文件放入 `apps/miniapp/src/static/fonts/` 并在全局样式中声明。

## 注意事项

- 不得提交来源不明或未明确授权的小程序内嵌字体文件。
- 若使用方正字体文件、云字库或在线 Web Font，应先确认授权范围覆盖微信小程序发布。
- 保留 `.vs-font-pixel` 作为历史工具类名兼容，但实际指向方正展示字体栈。
