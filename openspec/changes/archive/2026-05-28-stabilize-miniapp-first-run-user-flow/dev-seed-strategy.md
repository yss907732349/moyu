## 开发演示种子策略

本变更不要求生产环境自动生成演示内容。日报、社区和生存账本的种子数据只允许用于开发、测试或演示环境，并且必须由开发者显式运行脚本或启用准备流程。

## 日报

开发环境可以创建一份满足发布规则的已发布日报，用于验证首页摘要、日报首页、栏目列表和文章详情。外部采编文章必须提供来源标题、来源站点和来源 URL，不得编造无法追溯的来源。

## 社区

开发环境可以创建少量 `approved` 示例帖子，用于验证列表和详情。公开响应仍必须遵守作者快照边界，不得暴露薪资、工作档案、生存账本、微信身份或后台审核内部字段。

## 生存账本

普通用户首跑不得自动创建外卖、下午茶或通勤假订单。开发验证应覆盖无订单时的 0 值摘要、空详情和 0 值本周报告，而不是依赖伪造消费通过。
