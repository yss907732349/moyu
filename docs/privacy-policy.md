# 摸鱼隐者隐私政策

版本：2026-06-08

本文档用于说明 `摸鱼隐者` 小程序在社区发布、内容安全、举报治理和后台运营中如何处理个人信息。实际发布前应结合微信小程序平台要求、公司主体信息和法务审核补齐正式条款。

## 信息收集范围

为提供基础功能，系统会处理以下信息：

- 微信登录相关标识：用于识别应用内用户和维持登录态，不在公开页面展示。
- 隐者档案：昵称、阵营、头像 key、等级、称号和徽章，用于社区作者展示。
- 工作档案：薪资、工作时间、隐藏模式等仅用于本人首页计算，不进入社区公开资料。
- 社区内容：帖子、评论、回复、图片、点赞、收藏、举报和审核状态。
- 手机号验证状态：用户发帖、评论或回复前，通过微信手机号验证组件完成验证；服务端只保存验证状态、验证时间、验证来源和必要哈希材料。
- IP 属地：用户成功发帖、评论或回复时，服务端从可信请求来源识别客户端 IP，并保存发布时降精度属地快照。
- 内容安全和举报治理记录：包括审核结论、风险标签、举报原因、处理动作和必要的后台审计记录。

## 手机号验证边界

手机号验证只用于社区发帖、评论和回复的发布门槛，以及后台合规状态判断。

系统不会在公开帖子、评论、回复、作者快照、通知、公开个人主页或普通资料页展示完整手机号、手机号尾号或“已验证手机号”徽章。后台常规治理页面只展示是否完成手机号验证的布尔状态和协议版本摘要，不展示完整手机号或手机号尾号。

微信登录 `code` 和手机号验证动态 `code` 是不同凭证，服务端不得混用。手机号验证失败时，用户保持未验证状态，系统不会绕过门槛代为发布。

## IP 属地展示边界

系统会在社区内容发布成功时保存发布时 IP 属地快照。公开侧最多展示境内省、自治区、直辖市或境外国家/地区，例如 `IP属地：广东`、`IP属地：日本` 或 `IP属地：未知`。

公开侧不会展示完整 IP、市、区县、街道、IP 哈希、代理 header 或解析原始结果。历史无属地快照的内容不会根据当前请求或客户端传值补造真实属地，只能隐藏该字段或展示 `未知`。

帖子详情、评论、回复和公开个人主页的 IP 属地展示位在提审包和生产包默认开启，普通用户不能关闭或自定义。

## 内容安全与举报治理

用户提交的昵称、帖子、评论、回复和社区图片会进入内容安全审核流程。系统可能使用本地敏感词、微信内容安全、AI 辅助审核和人工复核。图片审核可能通过异步回调完成，在结果确认前内容可能仅作者本人可见或进入人工复核。

用户可以举报公开帖子、评论和回复。系统会保存举报目标、结构化举报原因、处理状态、有效举报风险和必要的后台操作审计。举报处理不会向被举报作者公开举报人的身份或补充说明。

后台运营人员可查看内容 ID、作者用户 ID、手机号验证布尔状态、协议版本、降精度 IP 属地、内容安全摘要和举报处理状态，用于审核、隐藏、移除、禁言、封禁或解除治理。后台常规响应不得包含完整手机号、明文 IP、微信 openid、unionid、sessionKey、登录 token、供应商密钥或供应商完整原始响应。

## 保存与删除

隐私同意版本、手机号验证状态、社区发布 IP 属地快照、内容安全结果、举报和治理审计会按社区运营、合规留痕和安全排查需要保存。社区内容被隐藏、移除或不再公开后，公开个人主页的最近 IP 属地只使用仍可公开内容中的最近一次发布记录。

用户可通过小程序或客服渠道反馈资料错误、内容处理异议和隐私问题。后续若增加申诉中心、实名核身、人脸识别、设备指纹或手机号尾号排查能力，应先更新本政策、社区用户协议和 OpenSpec change。
