# MVP 人工内测检查清单

人工检查用于覆盖脚本不适合稳定模拟的浏览器实时交互、微信开发者工具、真机 API 地址、真实转链、真实订单回流和真实 AI 审核。以下外部服务项不属于默认确定性总 smoke 的通过前提。

## 失败记录规则

- 记录检查项、角色、环境、时间、操作路径、截图或录屏、接口路径和状态码。
- 不记录后台 token、完整手机号、手机号尾号、明文 IP、微信 openid/unionid/sessionKey、微信 `access_token`、微信内容安全完整 `trace_id`、聚推客 `apikey`、完整 `sid`、完整订单原始响应、AI 原始 prompt、供应商完整原始响应或真实用户隐私字段。
- 外部服务失败不阻断默认 smoke，但必须在内测风险列表中登记。
